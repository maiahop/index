BSIT EVOTE: Online Voting System

Category: Online System | Group Project

Project Details

  • Role: Designer, System Developer
  • Tools: C# .NET, Visual Studio, Windows Forms Application
  • Database: SQL Server
View Project Repository →

BSIT EVOTE is a desktop-based online voting system made for BSIT students. It was developed using C# Windows Forms and aims to make student elections easier, faster, and more secure.

Key Features: Secure voter authentication, real-time vote counting and tracking, candidate management, and an intuitive user interface for a seamless voting experience.
